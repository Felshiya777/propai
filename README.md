# 🏘️ PropAI — AI-Powered Real Estate Intelligence

India's smartest property price predictor with dark luxury UI, ML models, and AI advisor.

## 🌐 Live Demo
> https://propai-jt4vavahqjnigeny777e4wrv.streamlit.app

## ✨ Features
- 🔑 Secure Login & Sign Up
- 🏠 ML Price Predictor (Random Forest)
- ⚖️ Compare Properties Side by Side
- 🔔 Price Alerts System
- 📍 Investment Hotspot Map (18 Indian Cities)
- 🤖 AI Property Advisor (Groq AI)
- 📸 Virtual Property Tour Analyzer
- 📊 Market Insights & News Feed
- 🌤️ Locality Report (Weather, Flood Risk, AQI)
- 🧮 Property Tax Calculator
- 🏦 EMI Calculator
- 📄 PDF Report Download
- 💬 Negotiation Tips

## 🛠️ Tech Stack
- Streamlit, scikit-learn (Random Forest), Groq AI, Plotly, pandas, Python

## 🚀 Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## 🔐 Secrets Required
Add to Streamlit Cloud secrets:
```toml
GROQ_API_KEY = "gsk_your_key_here"
```
